# Lesson Template: Egyptian Arabic Lessons

## Lesson Structure Overview

Each lesson follows this structure:
- **Lesson Title** (Arabic, English, Transliteration)
- **Level** (beginner/intermediate/advanced)
- **SubLessons** (multiple sub-lessons that break down the main topic)

Each SubLesson contains:
1. **Content Section** - New vocabulary, phrases, and grammar explanations
2. **Practice Exercises** - Interactive exercises to reinforce learning
